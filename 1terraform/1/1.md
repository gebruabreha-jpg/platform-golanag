Prerequisites:-
Terraform (CLI) — yes, must be installed. You clearly already have it (your terraform plan ran).

azurerm provider — no, you don't install it manually. terraform init already downloaded it (that's the .terraform.lock.hcl file). Terraform fetches providers automatically.

Azure CLI (az) — only needed for az login auth. Not required if you use a Service Principal via env vars instead.

#  Install Azure CLI and login
Install from https://learn.microsoft.com/cli/azure/install-azure-cli-windows
az login

# Set subscription
az account set --subscription "<your-subscription-id>"

# Verify Terraform installed
terraform version
